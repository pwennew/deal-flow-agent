## Recommendations

### For Improving Web Search Coverage

1. **Run periodic searches** (weekly) vs. one-time snapshot
2. **Use Bloomberg Terminal** or Pitchbook for better access
3. **Monitor PE firm press release feeds** directly
4. **Track SEC filings** (8-K for material events)
5. **Expand to DealBook, Mergermarket, Private Equity Wire**

### For Capturing Missing Signal Types

**PE Bid Submitted:**
- Search for "submitted bid" + company name
- Monitor activist investor 13D filings
- Track auction advisers (Lazard, Moelis, Evercore mentions)

**PE Interest:**
- Requires Bloomberg/Reuters terminal access
- Monitor "sources say" articles
- Track PE firm conference presentations

**Adviser Appointed:**
- Search for "mandated", "appointed adviser", "running sale process"
- Monitor investment bank earnings calls
- Track "league table" announcements

### For Deal Flow Agent Enhancement

1. **Add web search as supplementary source** - Run weekly searches for gaps in RSS coverage
2. **Create signal progression tracking** - Link Strategic Review → Adviser → Bid → Talks → Agreement → Close
3. **Build PE firm heatmap** - Track which firms are active in which sectors
4. **Implement valuation benchmarking** - Compare disclosed deal multiples by sector
5. **Add complexity scoring model** - Auto-classify based on employee count, geography, integration level
